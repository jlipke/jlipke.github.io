With this, they provide a customer website to allow customers and employees to use multiple accounts with varying permissions to retrieve/store customer's calibration data for auditing purposes.

Utilizing MVVM structure, and various RestAPI methods. It's UI was assembled with XAML with the focus on low resource iOS/Android devices.

It also managed Microsoft Outlook and Google API services.
